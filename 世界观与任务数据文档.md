## 一、世界观概述

### 1.1 时代背景

> **大唐年间，妖魔四起，无数百姓处于水深火热当中！**

- 十万天兵天将与各路妖魔战斗难解难分。
- 不断有英雄侠女从亿万百姓中脱颖而出，踏上降妖除魔的征程。
- 玩家从东海渔村（新手村）起步，逐步融入这个西游世界。

### 1.2 游戏特色（源自游戏内 NPC 指引说明）

| 特色 | 说明 |
|------|------|
| **类型** | 古典神话类手机网游，自动任务/自动打怪 |
| **门派** | 五大门派：将军府、方寸山、龙宫、月宫、普陀山 |
| **中心城** | 以长安城为中心城市 |
| **区域类型** | 天宫、地府、十八层地狱、水帘洞、取经路线诸国、副本/挑战/阵法区域 |
| **特殊区域** | 恶魔广场、洞天福地（万人云集处） |
| **装备系统** | 稀有装备、套装、结婚饰品、法宝、坐骑，装备合成/升级/打孔镶嵌 |
| **道具系统** | 自动回复药品、传送卷轴、召唤神仙符咒、多倍经验祝福道具、永久仙丹 |
| **任务类型** | 主线剧情、转职任务、区域任务、每日任务、隐藏任务、护送任务、挑战任务、副本任务、答题任务 |
| **竞技赛事** | 大型攻城战、全服国家帮派争霸大赛、全服个人天下争霸比武擂台 |

### 1.3 开局剧情

新角色创建后经历一套全门派介绍的选择流程（`xy295.php`）：

1. 开场语：大唐年间，妖魔四起...
2. 天兵天将与妖魔大战
3. 英雄侠女从百姓中挺身而出
4. 依次介绍五大门派背景，玩家选择加入
5. 起始地点：东海渔村（新手村）

## 六、任务系统总览

### 6.1 任务分类

```

├── 主线任务 (zxrw)     — 主线剧情，5个章节 × 100进程点
```
