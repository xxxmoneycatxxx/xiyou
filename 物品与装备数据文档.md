---

## 一、数据存储架构

所有物品/装备数据**硬编码在 PHP 文件**中，数据库 `wpxx` 表仅用于运行时查询/缓存。

| 数据类型 | 存储路径 | 文件数 | 数量 |
|----------|----------|--------|------|
| 物品信息 | `fqxy/wp/wpxx01~12.php` | 12 | ~1,020 个 |
| 装备信息 | `fqxy/wp/zbxx01~08.php` | 8 | ~700+ 件 |
| 物品使用效果 | `fqxy/wp/wpsy01~12.php` | 12 | 对应物品 ID |
| 装备锻造数据 | `fqxy/zbdz/` | 若干 | 合成/升级/打孔 |

### 物品属性结构

```php
$wpmz = "物品名字";
$wpms = "描述";
$wpjd = 2;    // 金豆价格
$wpjg = 1000; // 银两价格
$wpdj = 1;    // 等级
$wpzl = 1;    // 重量
$wpbd = 1;    // 绑定 (0=绑定, 1=不绑定)
$wpfl = 1;    // 分类编号
```

### 装备属性结构

```php
$wp1  = "装备名字";    $wp2  = "装备描述";
$wp14 = HP;            $wp15 = 攻击;
$wp16 = 魔攻;          $wp17 = 防御;
$wp18-20 = 冰/火/雷攻;  $wp21-23 = 冰/火/雷防;
$wp24 = 等级;          $wp25 = 重量;
$wp26 = 绑定;          $wp27 = 价格;
$wp28 = 佩戴;          $wp29 = 门派;
$wp30 = 部位分类;
```
