## 二、NPC/怪物属性公式

### 早期 NPC（编号 1~40 左右）：固定属性
新手村及早期 NPC 的属性采用**手工硬编码**，没有统一公式。

### 中后期 NPC（编号 54 起）：动态公式
属性由等级 `$ndj` 通过公式自动计算：

```
血量   HP = ndj × 2000
攻击   攻击 = ndj × 211
魔攻   魔攻 = ndj × 222
防御   防御 = ndj × 232
魔防   魔防 = ndj × 242
冰/火/雷攻 = ndj × 1
冰/火/雷防 = ndj × 1
MP     = ndj × 50
```

**示例**：等级 80 的 NPC
| 属性 | 值 |
|------|-----|
| HP | 160,000 |
| 攻击 | 16,880 |
| 魔攻 | 17,760 |
| 防御 | 18,560 |
| 魔防 | 19,360 |

**特殊 NPC（如李世民、BOSS等）**：采用定制属性，HP 可达 `ndj × 2,000,000`。

## 八、战斗系统

### 遇怪机制
- 玩家在地图中移动时，可能遭遇随机怪物
- 每次最多同时遇到 **3 只怪物**
- 怪物属性存储在玩家缓存 `ache/{wjid}/npc.ini` 中

### 战斗属性（npc.ini 缓存格式）
```ini
[怪物编号]
1号 = 怪物ID
2号 = 怪物ID
3号 = 怪物ID

[怪物1号属性]
hp = 血量
mp = 法力
攻击 = 物理攻击
魔攻 = 魔法攻击
防御 = 物理防御
魔防 = 魔法防御
冰攻 = 冰系攻击
火攻 = 火系攻击
雷攻 = 雷系攻击
冰防 = 冰系防御
火防 = 火系防御
雷防 = 雷系防御
```

### 伤害类型
游戏支持 4 种伤害计算维度：
- **物理伤害**：攻击 vs 防御
- **魔法伤害**：魔攻 vs 魔防
- **元素伤害**：冰攻/火攻/雷攻 vs 冰防/火防/雷防（三系独立计算）
